# 太空防御战 (Space Defense)

基于植物大战僵尸玩法，融合太空杀角色的高科技塔防游戏。

- [x] 高科技主题UI优化

### 🎮 当前功能

- **BootScene**: 启动场景，带进度条的加载界面
- **MenuScene**: 太空主题主界面，包含航母、核弹、太空飞船等元素
- **GameScene**: 完整的铁板网格系统和防御塔放置
- **GridSystem**: 逼真铁板可视化网格，支持交互放置
- **DevTools**: F1调试模式，性能监控

## 🛠 技术栈

- **游戏引擎**: Phaser.js 3.70.0
- **开发工具**: Vite 4.4.0
- **语言**: ES6+ JavaScript
- **样式**: CSS3

## 📦 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
